Welcome to the Commerce.js docs. This monorepo ships a Next.js 16 storefront, standalone checkout, admin dashboard, and shared packages for payments, storage, and catalog data.

## What is Commerce.js?

Commerce.js is a **commerce-agnostic** toolkit: swap adapters, payment providers, and storage backends without rewriting your storefront.
