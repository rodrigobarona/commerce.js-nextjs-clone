## Prerequisites

- Node.js 24+
- pnpm 10

## Install

```bash
pnpm install
cp .env.example apps/storefront/.env.local
cp .env.example apps/checkout/.env.local
pnpm db:setup
pnpm dev
```

## Dev servers

After `pnpm dev`, Turbo starts all apps:

- Marketing: [http://localhost:3001](http://localhost:3001)
- Storefront: [http://localhost:3000](http://localhost:3000)
- Dashboard: [http://localhost:3002](http://localhost:3002)
- Docs: [http://localhost:3003/docs](http://localhost:3003/docs)
- Checkout: [http://localhost:3004](http://localhost:3004)

Env vars live in the repo root `.env.local`. Run `pnpm env:link` once so every app symlinks to it.

## Adding UI components

Shared shadcn primitives live in `@prood/ui`:

```bash
pnpm dlx shadcn@latest add button -c packages/ui
```

Commerce-specific components are in `packages/ui/src/components/*`.
