`@prood/types` is the **shared vocabulary** for all Prood packages. It contains TypeScript interfaces and types only — no runtime code, no dependencies.

## Installation

```bash
pnpm add @prood/types
```

Already included as a dependency of `@prood/commerce`, `@prood/platform`, and `@prood/ui`.

## Domain types

### Catalog

```ts
import type {
  Product,
  ProductVariant,
  ProductOption,
  Category,
  Brand,
  SearchResult,
  SearchParams,
} from '@prood/types'
```

### Cart & checkout

```ts
import type {
  Cart,
  CartItem,
  CartTotals,
  ShippingMethod,
  PaymentMethod,
  Address,
  CustomerInfo,
} from '@prood/types'
```

### Orders

```ts
import type {
  Order,
  OrderItem,
  OrderStatus,
  FulfillmentStatus,
} from '@prood/types'
```

### Store & customers

```ts
import type { StoreInfo, Customer, Country } from '@prood/types'
```

## Adapter interfaces

```ts
import type {
  CommerceAdapter,
  CatalogAdapter,
  CartAdapter,
  CheckoutAdapter,
  OrdersAdapter,
  CustomersAdapter,
  StoreAdapter,
} from '@prood/types'
```

Implement these to create a custom commerce backend adapter.

## Error handling

```ts
import { CommerceError, toErrorResponse, priceToMajorAmount } from '@prood/types'

// Throw structured errors
throw new CommerceError('Product not found', 'NOT_FOUND', 404)

// Convert to HTTP response
const { error, code, status } = toErrorResponse(err)

// Price formatting
priceToMajorAmount(9999, 'EUR') // → 99.99
```

## Package structure

```
packages/types/src/
├── index.ts           # Re-exports everything
├── product.ts         # Product, Variant, Option
├── cart.ts            # Cart, CartItem, Totals
├── order.ts           # Order, OrderItem, Status
├── customer.ts        # Customer, Address
├── store.ts           # StoreInfo
├── adapter.ts         # CommerceAdapter interfaces
├── payment.ts         # PaymentProvider interface
├── storage.ts         # StorageProvider interface
├── error.ts           # CommerceError, toErrorResponse
├── admin.ts           # Admin input/output types
└── specialized/       # Auction, rental, wholesale, etc.
```
